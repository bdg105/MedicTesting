Программа для тестирования. Изначально создавалась для тестирования по медицинской тематике, поэтому такое название и приведён такой набор вопросов. Далее описана структура файла с вопросами.

Расшифровка quest_*.txt
	на месте * - указывается номер части с вопросами, кол-во частей в программе пока зашиты в коде 

	Вопрос состоит из 6 строк:
	
		1-я строка может быть: 
			1~ - один вариант ответа
			2~ - несколько вариантов ответа
	
		2-я строка номер вопроса с точкой, и сам вопрос:
			1. Работает программа?
			2. Проверка 2-х вариантов!
		
		3-я - 6-я строки это варианты ответов:
			варианты с + означают правильные ответы:
				+ первый
				второй
				+ третий
				+ четвёртый
